# Markdown to HTML Converter

A lightweight command-line tool to convert Markdown files into styled HTML documents. Uses a custom regex-based parser, so no external libraries like `markdown` or `pandoc` are required.

## Features

*   **Zero Dependencies**: Written in pure Python.
*   **Automatic Styling**: Generates a self-contained HTML file with basic CSS for readability.
*   **Syntax Support**: Handles headers, bold, italics, code blocks (inline), links, and lists.

## Usage

```bash
python run_converter.py [input.md] [output.html]
```

### Example

```bash
python run_converter.py README.md index.html
```

## Requirements

*   Python 3.x
